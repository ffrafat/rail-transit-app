# Play Store Release Notes

---

## v2.2.1 — Bug Fix Release

### 🇬🇧 English
- **Bug fix**: Fixed an issue where the app incorrectly showed an "update available" notification every day, even after the schedule data had already been updated. Users will no longer see false update prompts.

### 🇧🇩 বাংলা
- **বাগ ফিক্স**: একটি সমস্যা সমাধান করা হয়েছে যেখানে সময়সূচি আপডেট করার পরেও প্রতিদিন ভুলভাবে "নতুন আপডেট পাওয়া গেছে" নোটিফিকেশন দেখাচ্ছিল। এখন থেকে আর অপ্রয়োজনীয় আপডেট নোটিফিকেশন আসবে না।
